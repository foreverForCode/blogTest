
## 入门

### 如何安装gitbook-cli

> 不建议安装最新版的gitbook-cli，有坑

```
npm install gitbook-cli@2.0.0 -g
```

### 如何初始化

```
gitbook init
```

> 它会生成 README.md 和 SUMMARY.md

README.md 是首页，SUMMARY.md 是指文章目录

### 动态预览

```
gitbook serve
```

> 由于每一次改动都会重新下载插件，有时候会失败，不建议使用

### 打包静态HTML

```
gitbook build
```

> 会成_book文件夹，所有的静态HTML文件都存放在这里
